=============================
Evaluating type-A uncertainty
=============================

Type-A evaluation of uncertainty involves the statistical analysis of data, by definition.
Type-B evaluation uses any means other than statistical analysis.

The shorter name ``ta`` has been defined as an alias for ``type_a``,
to resolve the names of objects defined in this module.
